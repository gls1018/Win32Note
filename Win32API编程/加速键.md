## 加速键

加速键就是快捷键. 
例如, 菜单项中的 `Save` 保存选项的快捷方式一般为 `Ctrl + S`
当用户按下 Ctrl + S 时, 会发送 `WM_COMMAND` 消息, 通过 `LOWORD(wParam) == IDM_FILE_SAVE` 来触发相应的代码逻辑

### ACCEL结构体

```cpp
typedef struct tagACCEL 
{
    BYTE   fVirt;  // 修饰键 + 类型              
    WORD   key;    // 键值
    WORD   cmd;    // 命令ID
} ACCEL, *LPACCEL;
```

- fVirt: 描述 按键类型 + 修饰键
  - FCONTROL : Ctrl
  - FALT     : Alt
  - FSHIFT   : Shift
  - FVIRTKEY : Key是虚拟键 (Vk_xxx)
  - FNOINVERT: 菜单不高亮,很少用

### CreateAcceleratorTable

```cpp
HACCEL CreateAcceleratorTableW(
  [in] LPACCEL paccel,
  [in] int     cAccel
);
```
- 作用: 注册加速键表.
- paccel: ACCEL类型数组指针
- cAccel: 数组中加速键条目的数量

### TranslateAccelerator

```cpp
int TranslateAccelerator(
  [in] HWND   hWnd,
  [in] HACCEL hAccTable,
  [in] LPMSG  lpMsg
);
```

作用: 将加速键翻译成 WM_COMMAND 消息

### 示例代码

```cpp

ACCEL accel[] = {
	{FCONTROL | FVIRTKEY, 'S',  IDM_FILE_SAVE },            // Ctrl + S  
	{FCONTROL | FSHIFT | FVIRTKEY, 'P', IDM_FILE_SAVE_AS }  // Ctrl + Shift + P
};         

HACCEL hAccel = CreateAcceleratorTableW(accel, 2);        // 创建 Accelerator Table.

MSG msg;
while (GetMessageW(&msg, hMainWindow, 0, 0))
{
	if (!TranslateAcceleratorW(hMainWindow, hAccel, &msg))   // 翻译加速键为 WM_COMMAND 消息
	{
		TranslateMessage(&msg);
		DispatchMessageW(&msg);
	}
}

DestroyAcceleratorTable(hAccel);  //别忘记销毁 AcceleratorTable.
```

首先创建一个 ACCEL类型的数组, 然后使用`CreateAccelertorTable`函数
