**Win32窗口程序的创建过程**

- 定义窗口 (WNDCLASSEX 结构体)
- 注册窗口 (RegisterClassEx)
- 创建窗口 (CreateWindowEx)
- 显示窗口  (ShowWindow)
- 更新窗口 (UpdateWindow)
- 消息循环
- 窗口过程 WndProc


**桌面程序的Main函数**

桌面程序的main函数声明如下:
```cpp
int  WinMain(HINSTANCE hIns, HINSTANCE hPreIns, LPTSTR lpCmdLine, int nShow)
```
- hIns: 当前程序实例句柄
- hPreIns: 无意义
- lpCmdLine
- nShow

**WNDCLASSEX 结构体**

```cpp
typedef struct _WNDCLASSEX {
    UINT     cbSize;             // 结构体的大小           //sizeof(WNDCLASSEX)
    UINT     style;              // 窗口样式
    WNDPROC  lpfnWndProc;        // 窗口处理程序
    int      cbClsExtra;         // 窗口类的附加数据大小    //0
    int      cbWndExtra;         // 窗口的附加数据大小      //0
    HANDLE   hInstance;          // 窗口类的应用程序实例句柄 
    HICON    hIcon;              // 窗口的图标
    HCURSOR  hCursor;            // 窗口的光标句柄
    HBRUSH   hbrBackground;      // 窗口的背景画刷
    LPCSTR   lpszMenuName;       // 窗口菜单名称
    LPCSTR   lpszClassName;      // 窗口类名称
    HICON    hIconSm;            // 窗口的缩略图标
} WNDCLASSEX;
```

- cbSize: 结构体大小, 必须设置为 sizeof(WNDCLASSEX)
- style: 常见的窗口样式如下:
  - CS_VREDRAW | CS_HREDRAW  ;  窗口的垂直方向和水平方向发生变化时，窗口会被重绘(V: Vertical   H:Horizontal)
  - CS_DBLCLKS:窗口可以接收双击消息, 用户双击时可以触发`WM_LBUTTONDBLCLK`或`WM_RBUTTONDBLCLK`消息** 
  - CS_NOCLOSE: 禁用窗口关闭按钮
- hInstance: 设置为 WinMain函数的 hIns.
- hCursor: 传参为 NULL, 默认是光标箭头
  - 作用: 当鼠标移动到窗口客户区时, 默认显示的光标形状.
  - 加载方式: 调用 `LoadCursor(nullptr, IDC_ARROW)`函数.
  - 几个默认值:
    - IDC_ARROW: 标准箭头
    - IDC_IBEAM: 竖杠/I形光标
    - IDC_WAIT: 沙漏/转圈圈光标
    - IDC_CROSS: 十字光标
    - IDC_SIZEALL: 四向箭头
    - IDC_HAND: 手形光标
- hIconsm: 窗口小型图标的句柄, `Icon  small`, 任务栏,窗口标题栏, 或者快捷方式上的图标
- lpszClassName: 窗口类名,窗口类的标识符,必须唯一


**CreateWindowEx 函数**

```cpp
HWND CreateWindowEx(
  DWORD     dwExStyle,       // 扩展样式
  LPCSTR    lpClassName,     // 窗口类名
  LPCSTR    lpWindowName,    // 窗口名称（标题）
  DWORD     dwStyle,         // 窗口样式
  int       x,               // 窗口左上角的 x 坐标
  int       y,               // 窗口左上角的 y 坐标
  int       nWidth,          // 窗口宽度
  int       nHeight,         // 窗口高度
  HWND      hWndParent,      // 父窗口句柄
  HMENU     hMenu,           // 菜单句柄
  HINSTANCE hInstance,       // 应用程序实例句柄
  LPVOID    lpParam          // 附加的窗口创建数据（可以是 NULL）
);
```

- dwExStyle: 扩展样式;
  - WS_EX_TOPMOST：窗口始终置顶
  - WS_EX_TRANSPARENT：窗口透明，鼠标点击可以穿透
  - WS_EX_LAYERED：分层窗口，支持透明度和颜色键
  - WS_EX_CLIENTEDGE：窗口客户区有凹陷边缘
  - WS_EX_WINDOWEDGE：窗口边缘凸起
  - WS_EX_ACCEPTFILES:窗口可以接收文件, 拖拽文件到窗口上,系统会发送WM_DROPFILES消息\
- lpClassName: WNDCLASS中lpszClassName的值.
- lpWindowName: 窗口标题
- dwStyle:  窗口样式，
  - WS_POPUP:弹出式窗口,没有标题栏和边框,只有客户区
  - WS_OVERLAPPED:重叠式窗口,自带标题栏

**消息循环**
一个简单的消息循环如下:
```cpp
MSG msg
while(GetMessage(&msg, nullptr, 0, 0))
{
    TranslateMessage(&msg);
    DispatchMessage(&msg);
}
```


**简单窗口程序代码**

```cpp
#include <windows.h>
#include <tchar.h>
LRESULT CALLBACK WindowProc(HWND hWnd, UINT msg, WPARAM wPara, LPARAM lPara)
{
    switch (msg)
    {
        case WM_DESTROY:
        {
            PostQuitMessage(0);
            return 0;
        }
    }
    return DefWindowProc(hWnd, msg, wPara, lPara);
}

int _tWinMain(HINSTANCE hIns, HINSTANCE, LPTSTR lpCmdLine, int nShow)
{
    WNDCLASSEX wc{ 0 };
    wc.cbClsExtra = 0;
    wc.cbSize = sizeof(wc);
    wc.cbWndExtra = 0;
    wc.hbrBackground = (HBRUSH)COLOR_WINDOW;
    wc.hCursor = LoadCursor(nullptr, IDC_HAND);
    wc.hIcon = nullptr;
    wc.hIconSm = nullptr;
    wc.hInstance = hIns;
    wc.lpfnWndProc = WindowProc;
    wc.lpszClassName = _T("MyWindowClass");
    wc.lpszMenuName = NULL;
    wc.style = CS_VREDRAW | CS_HREDRAW;
    RegisterClassEx(&wc);
    //创建窗口
    DWORD dwExStyle = WS_EX_ACCEPTFILES;
    DWORD dwStyle = WS_OVERLAPPEDWINDOW;
    HWND hMainWnd = CreateWindowEx(dwExStyle, _T("MyWindowClass"), _T("WinApp"), dwStyle, 500, 700, 1500, 1000, NULL, NULL, hIns, NULL);

    ShowWindow(hMainWnd, SW_NORMAL);
    UpdateWindow(hMainWnd);
    //消息循环
    MSG msg;
    while (GetMessage(&msg, nullptr, 0, 0))
    {
        TranslateMessage(&msg);
        DispatchMessage(&msg);
    }
    return 0;
}
```
